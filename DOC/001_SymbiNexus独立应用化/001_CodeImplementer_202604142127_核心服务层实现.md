# 001 核心服务层实现

## 任务概述
实现 SymbiNexus Flutter 项目的7个核心服务层文件。

## 完成状态
全部完成，VS Code 诊断 0 错误。

## 实现文件清单

### 1. PivotService (枢轴偏移服务)
- 文件: `lib/core/services/pivot_service.dart`
- 3点枢轴偏移逻辑: 0:00-3:00 逻辑日期为前一天
- 方法: `getLogicalDate()`, `getLogicalDateStr()`

### 2. ChannelNormalizer (渠道归一化服务)
- 文件: `lib/core/services/channel_normalizer.dart`
- 11条支付渠道别名映射 (17个别名 -> 10个标准名)
- 方法: `normalize()`, `standardChannels`

### 3. LlmRouter (LLM路由服务)
- 文件: `lib/core/services/llm_router.dart`
- 8个LLM提供商配置 (zhipu/minimax/qwen/mimo/google/doubao/aliyun/longcat)
- 支持 OpenAI 和 Anthropic 两种协议
- 方法: `getAvailableProviders()`, `callProvider()`, `callWithFallback()`
- API密钥从 flutter_secure_storage 读写

### 4. AiPipeline (AI管道服务)
- 文件: `lib/core/services/ai_pipeline.dart`
- 3阶段管道: 分类 -> 语义优化 -> 结构化提取
- 11个模块分类，4个模块禁用Stage2
- 每个模块独立提取提示词，MVP阶段硬编码
- JSON响应容错解析

### 5. FrontmatterParser (增强)
- 文件: `lib/core/markdown/frontmatter_parser.dart`
- 使用 yaml 包解析 (替代 _raw)
- 容错: 有引号/无引号值、日期自动解析、列表值
- 新增: `serialize()`, `compose()` 序列化方法
- 降级: YAML解析失败时逐行容错解析

### 6. MarkdownIo (增强)
- 文件: `lib/core/markdown/markdown_io.dart`
- 支持自定义 vaultPath
- 自动处理 frontmatter 读写
- 新增: `readWithFrontmatter()`, `writeWithFrontmatter()`, `updateFrontmatter()`, `exists()`, `deleteFile()`, `listFiles()`

### 7. Providers
- 文件: `lib/core/providers/providers.dart`
- 6个 Riverpod Provider: pivotService, channelNormalizer, llmRouter, aiPipeline, database, markdownIo
