三步快速上手
STEP 01 · 安装
npm i -g @anthropic-ai/claude-code
全局安装，需要 Node.js 18+
STEP 02 · 进入项目
cd your-project && claude
在项目目录启动，自动读取上下文
STEP 03 · 开始对话
> 帮我重构 auth.ts 的登录逻辑
用自然语言描述任务，直接开干
💡 核心理念：Claude Code 直接读取你的代码库，理解整个项目结构后再执行任务。无需粘贴代码片段 —— 直接说「帮我找所有用了 deprecated API 的地方并修复」即可。
启动参数 / CLI Flags
会话模式
$
claude 交互式 REPL 模式（默认）
$
claude --continue 继续上一次会话
$
claude --resume 打开历史会话选择器
$
claude --from-pr 123 从指定 PR 恢复会话
非交互 / 管道模式
$
claude -p "修复所有 TS 类型错误" 单次执行后退出（适合脚本/CI）
$
cat error.log | claude -p "分析这个错误" 通过管道传入上下文
$
claude -p "..." --output-format json 输出 JSON（方便脚本解析）
$
claude --headless 无头模式，适合服务器环境
会话内斜杠命令 / Slash Commands
/help
查看所有可用命令
/clear
清空对话历史，释放上下文
/rewind
时间回溯，还原代码/对话
/resume
切换到另一个历史会话
/compact
压缩历史，减少 token 占用
/context
查看当前 token 使用情况
/vim
切换为 vim 键位输入模式
/config
查看或修改配置项
/model
切换使用的 Claude 模型
/mcp
管理 MCP 服务器连接
/init
生成 CLAUDE.md 项目说明文件
/exit
退出 Claude Code
