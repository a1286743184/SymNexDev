

## 1. 核心原则
* **差异化对待：** 成犬（有心理阴影）重在“修复与接受”，幼犬（白纸一张）重在“脱敏与预防”。
* **工具选择：** 不强求统一工具，以“狗不应激”为最高标准。

## 2. 月饼（成犬/PTSD修复）
* **现状诊断：** 极度抗拒指甲钳（曾出血/吓尿），对吸尘器/打印机噪音敏感，但接受吹风机。
* **禁止事项：** * ❌ 停止使用传统指甲钳（避免加深创伤）。
    * ❌ 停止送去宠物店强行修剪（避免由于保定导致的二次伤害）。
* **执行方案 A（优选-电动磨甲）：**
    * **工具：** 宠物专用电动磨甲器（静音款 + 金刚石磨头 + LED灯）。
    * **技巧：** “丝袜隔离法”（防卷脚毛）+“单趾固定法”（减少震动传导/止痒）。
    * **注意：** 需先进行声音测试（确保不触发吸尘器PTSD），少量多次，避免发热。
* **执行方案 B（保底-物理磨甲）：**
    * **工具：** 自制或购买的磨甲板（贴80目粗砂纸的木板）。
    * **玩法：** 利用零食诱导其主动抓挠板子，完全无强制、无噪音、无震动。

## 3. 云腿（幼犬/脱敏预防）
* **黄金法则：** 必须自己动手，掌控节奏，将剪指甲变成“信任游戏”。
* **脱敏SOP（每日每顿）：**
    1.  **触碰练习：** 捏爪子 -> 奖励；捏指头 -> 奖励。
    2.  **工具联想：** 指甲钳碰指甲（金属触感） -> 奖励；空剪一声（听声音） -> 奖励。
    3.  **实操节奏：** 每天只剪 **1** 个指甲（哪怕只剪空气或尖端），剪完立刻给大奖并结束。
* **外出策略：** 去宠物店仅做“洗脑”练习（上美容台吃零食），不进行实质性修剪，建立美好联想。

## 4. 意外处理
* 若出血，保持镇定，使用止血粉，并立刻给予超级大奖（覆盖痛觉记忆）。
