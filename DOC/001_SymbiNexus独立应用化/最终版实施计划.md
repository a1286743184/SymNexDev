## 一、审核发现的关键问题与修正

### Phase 3: 平台集成 (3.5周)

| 任务 | 内容 | 时间 | 修正说明 |
|------|------|------|---------|

| P3-02 | 快速录入Widget | 3天 | |

**P3-01修正**: 通知系统改用AlarmManager
- 精确通知: AlarmManager.setExactAndAllowWhileIdle()
- 周期性通知: AlarmManager + BroadcastReceiver
- 通知操作按钮: snooze(延迟10分钟) / complete(标记完成)
- 通知渠道: tasks(高) / memos(高) / routines(默认) / bills(默认) / sync(低)

**P3-03修正**: Syncthing同步改用SAF轮询
- Android: SAF(Uri) + WorkManager 5分钟轮询
- Windows: watcher包文件系统监听(Windows无SAF限制)
- REST API作为辅助监控(检测同步完成事件)
- 冲突检测: 扫描.sync-conflict文件，提示用户手动解决
